# VoiceAgent QA Platform

Comprehensive testing and monitoring suite for voice and chat AI agents. Provides automated conversation simulation, performance monitoring, and quality assurance capabilities.

## Features

1. **Conversation Flow Testing** - Multi-turn dialogue simulation with context retention
2. **Voice Quality Assessment** - Speech recognition and TTS analysis
3. **Agent Performance Monitoring** - Real-time metrics and alerting
4. **Regression Testing Suite** - Golden dataset comparison and CI/CD integration
5. **Scenario-Based Load Testing** - Concurrent conversation simulation under load

## Tech Stack

- **Frontend**: React 19 + Vite + Tailwind CSS
- **Backend**: FastAPI with Python 3.11+
- **Database**: SQLite
- **Testing**: Playwright + pytest

## Getting Started

```bash
chmod +x init.sh
./init.sh
```
